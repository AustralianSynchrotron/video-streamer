# video-streamer
Video streamer for MXCuBE


### Installation

Create a seperate environment for the depdencies:

```
git clone https://github.com/mxcube/video-streamer.git
cd video-streamer

# optional 
conda env create -f conda-environment.yml

# For development
pip install -e .

# For usage 
pip install .
```
